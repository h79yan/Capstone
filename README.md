# Que Food Hall - AI-Powered Food Ordering Solution

**Que Food Hall** is a machine-learning-powered platform that personalizes food ordering based on health statistics and streamlines restaurant operations.

### Live Demo
[Watch the Demonstration on YouTube] (https://www.youtube.com/watch?v=oquNnZy9Erw)

### Core Tech Stack
* **Frontend/Mobile:** iOS, Expo Go
* **Backend:** Python Microservices
* **AI/ML:** LLM integration for personalized recommendations

### Architecture Highlights
- Integrated a **Language Model** to analyze user health data and customize menus.
- Built a **real-time merchant platform** for scalable order management.
